Welcome to Squaak!

Squaak is a Real-time communication application.

Squaak is a Web-application with video chat and real-time messaging capabilities.

Squaak's Video Chat utilizes Agoras SDK and Agora Token Builder.

Squaak's Messaging utilizes Django Channels and Web Sockets in order to provide RTC.

The friends interface allows users to interact with one-another in various ways.

The notifications in this web-app utilize Django notifications hq.

Friend System

User May:

  • Add or Remove potential friends.
  • Cancel their pending friend requests.
  • They may also decline incoming friend requests.

Video Chat

User May:

  • Initiate 1 to 1 Video chat rooms with their friends.
  • Create Group Video chat rooms.

Real-time Messaging

User May:

  • Initiate 1 to 1 Direct Messaging with their friends.
  • Create / join Public Real-time chat rooms.
